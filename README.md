# InsightU – Student Learning Web Application

## Theme
InsightU is an interactive educational web application designed to help students explore useful learning resources and improve their study experience.

## Purpose
The purpose of this web application is to provide a simple and interactive platform where students can access learning information, explore different features, and improve their knowledge.

## Features
- Responsive website design using HTML, CSS, and Bootstrap
- Navigation bar for easy page access
- Interactive image slider
- Form validation for user inputs
- Smooth scrolling and hover effects
- Dynamic content using JavaScript

## Pages
1. Home Page – Introduction to InsightU
2. Dashboard / Features Page – Displays learning features and resources
3. Login Page – Allows users to log in
4. Contact Page – Allows users to send messages

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Use
1. Open the `index.html` file in a web browser.
2. Navigate through the pages using the navigation bar.
3. Explore the features and interactive elements of the website.
