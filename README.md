# 🎓 InsightU Website – Fullsite

## 📌 Project Overview

**InsightU** is an interactive web application developed to support students in managing their learning activities efficiently. The system provides user authentication, a personalized dashboard, and a clean, responsive interface.

This project was created as part of the **COM 2303 – Web Design** course to demonstrate practical web development skills using modern technologies.

---

## 🎯 Objectives

* Develop a responsive web application using Bootstrap
* Implement user authentication with PHP
* Create interactive features using JavaScript
* Design a clean and user-friendly interface
* Host the project on GitHub

---

## 🚀 Features

### 👤 User Authentication

* User Signup and Login system
* Secure session handling
* Redirect to dashboard after login

### 📊 Dashboard

* Personalized user dashboard
* Organized layout with sidebar and navigation
* Easy access to features

### 🌐 Pages Included

* Home Page (`index.php`)
* Contact Page (`contact.php`)
* Login Page (`login.php`)
* Signup Page (`signup.php`)
* Dashboard (`dashboard.php`)
* Profile Page (`profile.php`)

### ⚡ JavaScript Interactivity

* Form validation
* Dynamic UI updates
* Event handling and interactive components

## ⚙️ Installation & Setup

1. Install **XAMPP / WAMP**
2. Place the project folder in:

   ```
   htdocs/
   ```
3. Start **Apache** and **MySQL**
4. Import the database file:

   ```
   insightu_db.sql
   ```

   using **phpMyAdmin**
5. Run the project:

   ```
   http://localhost/InsightU
   ```
